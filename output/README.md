# Analysis Output

This directory contains analysis results and visualizations.

## Files

### analysis_output.txt
Dataset statistics and summary:
- Feature distributions
- Missing value counts
- Data type information
- Memory usage

### correlation_results.txt
Correlation analysis:
- Feature correlation matrix
- Strong correlations (>0.7)
- Feature relationships
- Statistical significance

### Visualizations
Generated by `f1_data_analysis.py`:
- Correlation heatmaps
- Distribution plots
- Scatter plots
- Time series analysis

## Usage

Regenerate outputs:
```bash
python src/f1_data_analysis.py
```

All outputs are automatically saved to this directory.
