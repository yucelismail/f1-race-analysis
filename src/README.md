# F1 Race Analysis - Source Code

This directory contains all the Python scripts for the data pipeline.

## Scripts

### 1. f1_database_enrichment.py
Enriches raw F1 data with F1DB historical database

### 2. f1_weather_enrichment.py
Fetches historical weather data from Open-Meteo API

### 3. f1_preprocessing.py
Prepares data for machine learning

### 4. f1_data_analysis.py
Data analysis and visualization
